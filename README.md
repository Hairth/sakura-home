# Sakura Home

一个参考 `jinghuashang.github.io` 与 `XinghuisamaBlogs` 视觉氛围制作的个人主页仪表盘。

## 内容

- 左侧个人导航
- 顶部快捷工具栏和日期
- 音乐、邮件、天气、时间、最近文档、收藏、VPS 状态模块
- 右侧樱花房间插画背景
- 轻量交互：实时时钟、音乐按钮、快捷操作提示

## 本地预览

直接打开 `index.html` 即可。也可以在项目目录运行：

```powershell
python -m http.server 4173
```

然后访问 `http://127.0.0.1:4173`。
