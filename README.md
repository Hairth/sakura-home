# Sakura Home

参考 `jinghuashang.github.io` 和 `XinghuisamaBlogs` 的视觉氛围制作的个人主页仪表盘。现在不是纯静态页面，而是一个可本地运行的前后端小应用。

## 已实现

- 文档：新建 Markdown、上传文件、下载上传文件、删除文档
- 收藏：新增收藏、打开链接、删除收藏
- 邮件：写入本地收件箱、查看详情、标记已读、归档
- 待办：新增、勾选完成、删除
- 笔记：记录灵感并写入本地数据
- 天气：通过 Open-Meteo 获取实时天气，失败时自动使用离线值
- 音乐：浏览器 Web Audio 本地合成音播放、暂停、切歌、喜欢状态持久化
- 设置：修改站点标题、副标题、欢迎语、短句、天气位置

## 本地运行

不需要安装依赖，直接运行：

```powershell
cd Z:\sakura-home
npm start
```

然后访问：

```text
http://127.0.0.1:4173
```

运行数据会写入 `data/db.json`，上传文件会放到 `uploads/`。这两个运行时内容默认不提交到 Git。

## 部署提示

之后部署到 VPS 时，把仓库放到服务器上，执行：

```bash
npm start
```

再用 Caddy / Nginx 反向代理到 `127.0.0.1:4173`。
